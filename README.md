# sounddeck-privacy

GitHub Pages source for the SoundDeck Android app's privacy policy.

The live page is generated from [index.md](index.md) and served by GitHub
Pages from the `main` branch. The URL is referenced from the Google Play
Console under **App content -> Privacy policy** for the SoundDeck app
(`com.henryhillproductions.sounddeck`).

## Editing

1. Edit `index.md` (Markdown with Jekyll front matter).
2. Commit to `main` -> Pages rebuilds automatically (~1 min).

The `_config.yml` pins the `jekyll-theme-minimal` theme so the page renders
with a clean two-column layout out of the box -- no custom CSS needed.
